### 📝 كيف تساهم
1. Fork المستودع
2. أنشئ فرع: `git checkout -b feature/my-feature`
3. أجرِ التعديلات
4. Commit: `git commit -m 'feat: وصف التعديل'`
5. Push: `git push origin feature/my-feature`
6. أنشئ Pull Request

### 🏷️ اصطلاحات Commit
| البادئة | الاستخدام |
|---------|-----------|
| `feat:` | ميزة جديدة |
| `fix:` | إصلاح خطأ |
| `docs:` | تحديث التوثيق |
| `style:` | تنسيق الكود |
| `refactor:` | إعادة هيكلة |
| `perf:` | تحسين الأداء |
